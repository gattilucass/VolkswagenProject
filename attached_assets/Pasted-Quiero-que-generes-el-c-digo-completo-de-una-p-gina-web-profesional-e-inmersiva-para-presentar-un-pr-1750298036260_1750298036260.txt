Quiero que generes el código completo de una página web profesional e inmersiva para presentar un proyecto conceptual de campaña publicitaria creada con IA Generativa. La campaña es para el lanzamiento del Volkswagen ID.4 bajo el lema “La Nueva Era Eléctrica”, con estética cyber-organic, tono emocional y un enfoque transmedia innovador.
La web se alojará en Vercel, por lo tanto el código debe estar preparado para deploy directo. Usar HTML5, TailwindCSS, y animaciones suaves con Framer Motion (o alternativa con JavaScript puro o Tailwind Plugins).

⚙️ Requisitos técnicos
Lenguaje: HTML + Tailwind CSS (sin frameworks como React, Svelte, etc.)

Estilo: Oscuro, elegante, minimalista, bien espaciado y con simetría.

Animaciones: Suaves, fluidas, al hacer scroll o hover.

Responsiva: Mobile-first, con diseño adaptable a todos los dispositivos.

Interactividad: Video embed, sliders, hover states, botones interactivos.

Preparado para agregar textos, imágenes y videos desde carpeta /assets.
